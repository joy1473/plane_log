# 경량항공기 비행기록 시스템 조사 (Phase 1)

## 한국교통안전공단 초경량 비행기록 시스템
- 탑승형 경량항공기용 개인 비행기록 실시간 API 공개 안 함
- 통계용 오픈 API만 존재 (공공데이터포털)
- Phase 1 현실적 방법 = 공단 초경량 비행기록 시스템 → CSV/엑셀 내보내기 → 수동 업로드

## 예상 CSV 열 (한국어 기준)
- 날짜, 비행 시작 시간, 비행 종료 시간, 비행 시간(분), 이착륙장, 교관 이름, 훈련 목적, 착륙 횟수, 비행 고도, 비고

## GPS 궤적 데이터
- GPS 궤적 데이터 포함 가능성 매우 낮음
- 포함 시 JSON 배열 또는 GeoJSON으로 저장 계획
